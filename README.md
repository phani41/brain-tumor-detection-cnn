# Brain Tumor Detection using CNN

A full-stack medical AI application for brain tumor classification from MRI images.

## Features
- CNN-based 4-class classification
- Flask REST API backend
- JavaScript frontend (no frameworks)
- Drag & drop image upload
- Confidence & probability visualization
- Uncertainty handling

## Classes
- Glioma
- Meningioma
- Pituitary
- No Tumor

## Tech Stack
- Python, TensorFlow, OpenCV
- Flask + Flask-CORS
- HTML, CSS, JavaScript
